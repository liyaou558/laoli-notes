> 基于 show notes 整理，非完整转录稿。本期克隆自 Invest Like The Best — Why the AI Boom Is Just Getting Started。

> Whale Rock Capital 创始人 Alex Sacerdote 从 S 曲线、护城河、盈利能力三个维度拆解 AI 投资。他认为企业 AI 渗透率不到 1%，全球算力已经不够用了——这不是泡沫初期，而是一条刚刚抬头的"L 曲线"。Anthropic 是他当前最高确信度的持仓，编程市场 5000 亿美元的翻盘机会正在被 Claude Code 撬动。以下为精简笔记。

> **核心观点**

> 1. **L 曲线刚刚开始**。企业 AI 渗透率不到 1%，全球算力已经出现结构性短缺。AI 总潜在市场 3-5 万亿美元，当前阶段不是泡沫末期，而是 S 曲线最底部的直线拉升段。

> 2. **基础模型寡头化**。从 60 多家竞争者收敛到三家——Anthropic、OpenAI、Google。三者差异明显：Anthropic 强在企业品牌与编程，OpenAI 强在消费者心智，Google 强在算力与分发。模型不是大宗商品，差异化只会扩大。

> 3. **Anthropic = 最高确信度**。企业级品牌信任、关键 IP 资产、模型的递归改进能力、编程领域领先优势、Claude Code 推动编程进入 Agentic 阶段——五重逻辑叠加，让 Anthropic 成为 Alex 最重仓的 AI 标的。

> 4. **AI 编码从 20% 翻转到 80%+**。Karpathy 曾预测 AI 写 20% 代码，新模型出来后比例反过来了。编程市场约 5000 亿美元，Agentic Coding 正在重新定义开发者工具链。

> 5. **软件公司面临四重死亡压力**。企业 IT 预算迁移到 token 消费、席位收费模型崩溃、招聘冻结波及工具采购、企业用 AI 自建替代品。"软件像马车，AI 像传送器"——Alex 几乎清仓了所有应用软件。

> 6. **护城河决定 S 曲线赢家**。RIM、Palm、Nokia 都曾站在移动 S 曲线上，但没有护城河，最终全输。数字世界的护城河是：网络效应、行业标准、规模、平台、关键 IP、品牌——Anthropic 每一项都有。

> 7. **硬件从商品变成战略性稀缺资源**。AI 工作负载每年 10 倍增长，高带宽内存、40 层 PCB 板、AI 服务器（每台 20-30 万美元）、数据中心光纤（一个数据中心绕地球四圈半）——整个硬件栈都在去商品化。

> 8. **AI 可以当记者，但不能当裁判**。Alex 每年见 2500-3000 场管理层，用 Scuttlebutt 方法交叉验证。"三脚架信心"系统——自己、分析师、外部投资人同时看好时下重注。AI 提升效率，但无法替代判断。

> **Q&A**

> **Q：什么是 L 曲线？为什么不叫 S 曲线？**
>
> 企业 AI 渗透率不到 1%，还在 S 曲线的最底部。从当下看，这条线几乎是直着往上拉的，所以 Alex 叫它 L 曲线。S 曲线的拐点需要"所有障碍都被移除"——就像 iPhone 发布时，触屏、应用商店、运营商套餐一次性到位，需求像火箭一样起飞。AI 的障碍正在逐个移除，但远未完成。

> **Q：基础模型市场最终剩下几家？**
>
> 从 60 家收敛到 3 家：Anthropic、OpenAI、Google。因为训练前沿模型需要天文数字的算力投入，资本门槛高到绝大多数玩家无法跨越。这三家有明显差异化——Anthropic 做企业深度，OpenAI 做消费者广度，Google 做基础设施整合。模型不是大宗商品，不会同质化。

> **Q：为什么 Anthropic 是最高确信度投资？**
>
> 五重逻辑：①企业品牌——在 CIO 圈子里建立了安全和可控的认知；②关键 IP——模型架构、训练方法、Claude Code 产品矩阵；③递归改进——模型越来越好，改进速度反超对手；④编程领先——Claude Code 是目前最强的 Agentic Coding 工具；⑤融资能力——能持续调用巨额资本维持前沿竞争。

> **Q：S 曲线的买入信号是什么？**
>
> 战略拐点。Alex 的框架是寻找"所有障碍同时被移除"的时刻——iPhone（触屏+应用商店+运营商）、Tesla（价格降到 3.5 万+续航 300 英里+供应链就位）、AWS（隐藏在 Amazon 内部的企业 IT 大曲线）。这些拐点靠直觉和模式识别，不是靠财务模型算出来的。错过早期不可怕，大 S 曲线持续很久。

> **Q：S 曲线的卖出信号？**
>
> 渗透率达到 30-40% 时。此时增速开始放缓，边际买家入场，竞争格局恶化。Alex 强调：有 S 曲线不等于能赢——RIM、Palm、Nokia 都曾站在智能手机的 S 曲线上，因为没有护城河，最后全部归零。

> **Q：数字世界的护城河长什么样？**
>
> 六要素：网络效应、行业标准、规模经济、平台锁定、关键 IP、品牌信任。Anthropic 目前在企业品牌和关键 IP 上领先，如果能建立起开发者生态的平台效应，护城河会进一步加深。反面教材是 RIM——有 S 曲线、没护城河，iPhone 一来就崩溃。

> **Q：AI 对传统软件公司意味着什么？**
>
> 四重压力同时袭来：①企业 IT 预算从软件许可迁移到 token/API 消费；②按席位收费的模式不再成立——AI 让一个人能干五个人的活，为什么还要买 100 个席位？③招聘冻结直接砍掉新工具采购；④企业用 AI 自建内部工具替代 SaaS。

> **Q：新版四十法则是什么？**
>
> 传统 SaaS 四十法则 = 收入增速 + 利润率 > 40%。Alex 的新版本：AI 收入占比 + 品类市场份额。如果一个软件公司的收入里 AI 相关部分不到两位数，且品类份额在下降，不管财务数字多好看都应该警惕。"软件像马车，AI 像传送器"——不是马车不够好，是传送器出现了。

> **Q：什么是"无界面软件"风险？**
>
> AI Agent 不再需要传统 UI。它们直接进入数据层，绕过前端界面完成操作。这意味着靠用户体验粘性建立的护城河可能一夜之间失效。当 Agent 可以直接读写数据库、调用 API、生成报表，谁还需要那个精美的 Dashboard？

> **Q：AI 编码市场有多大，谁会赢？**
>
> 全球编程市场约 5000 亿美元，AI 编码从辅助写 20% 代码翻转到自主写 80%+。Claude Code 是这个领域的领先者——它不是代码补全工具，而是能理解需求、规划架构、执行全链路开发的 Agent。这场变革比云计算更大，因为它改变了"谁在写代码"这个根本问题。

> **Q：硬件为什么不再是大宗商品？**
>
> AI 工作负载每年增长 10 倍，这彻底改变了硬件的供需结构。高带宽内存从通用 DRAM 变成关键瓶颈，AI 服务器每台 20-30 万美元且供不应求，网络设备从七年一换变成每年升级，PCB 板做到 40 层电路——整个供应链 30% 的缺口（DRAM、NAND、PCB）。Corning 光纤在单个数据中心里能绕地球四圈半。Celestica 从普通合同制造商变成了 Google TPU 服务器的关键供应商——这就是去商品化的缩影。

> **Q：Alex 的个人研究方法？**
>
> 三条腿走路（三脚架信心）：①自己深入研究，②内部分析师交叉验证，③外部投资人网络确认信号。每年 2500-3000 场管理层会议，用 Scuttlebutt 方法（见客户、供应商、竞争对手）拼图。他写了 90 页的 Claude Code 研究报告——用 Claude Code 帮助完成的。"AI 是很好的记者，但不能当裁判。它能帮你收集信息、整理资料，但判断未来只能靠人的判断力。"

> **Q：错过 AI 投资现在还来得及吗？**
>
> 完全来得及。企业 AI 渗透率不到 1%，这是几十年一遇的大 S 曲线。但 Alex 提醒：世界不会按指数级思考，大多数人太关注下一个季度的数字。真正的机会在于看准 5-10 年的结构性变化，找到有护城河的公司，然后坐稳。
